通过 Github 的操作来说明工作区、暂存区、版本库之间的关系

1. git工作流三者关系

工作目录,暂存区(Index),版本库(HEAD)三者关系如图:

2.在git命令行下的操作

  1. 创建一个文件夹后,在文件夹中git init,此目录为工作区.或直接git clone 一个项目,项目的目录为工作区
  2. 在文件修改变动后git add即把变动文件转到到暂存区
  3. 最后git commit后即提交到了版本库

3. 在git hub上的操作

3.1添加到工作区

  1. 点左上角上+后如图所示,增加相应git工作区.
    Add 为添加本地已经存在的文件夹,选中相应的文件夹后,Add repository即可

    Create 为本地的文件夹创建新的git工作区.

    Clone主要是将自己github上的项目下载下来.

    3.2添加到暂存区和版本库

  2. 新建git本地工作区后,到相应的文件夹下修改文件.如创建一个readme.txt的文件.github客户端自动会识别已经修改的内容,并自己勾选增加到暂存区(git add),如不加入可以不勾选.下面commit即为git commit的操作.

    3.3github客户端将两个操作合并成一个界面.

    在提交前前后用git for windows的git bash查看状态git status如下图:
    提交前提示需要git add,提交后直接就完成提交.相当一次完成两个操作.

3721-10对回顾的看法

如果你做一天的回顾，那么你在制定一天的计划时就非常有自信，因为你知道过去的一天是怎么度过的，有多长时间可以使用
同样的，如果你做了一周的回顾，你就会对一周的计划制定比较有把握，知道自己一个周的时间长度大约可以做多少事情，减少了盲目性。
因此说，回顾是计划的基础和保障，没有回顾的计划就像无源之水，不可持续
所以说， 3721 活动看似简单，只要求每天早晨列写三件事，每天晚上做回顾，正是因为要求每天必须做回顾，让大家对自己一天的精力投放做了回看，第二天就可以立马进行改进，每天改进一点点， 21 天就会有很大的变化

(1) 、谈谈自己对回顾的看法

记得在一个项目管理课听到，控制论有两种基本的方法，拿一个导弹命中目标的过程来说，有如下两种方法：
a. 收集关于距离、位置、湿度、风速等影响是否命中的相关因素，尽量量化这些因素并通过一定的模型来计算发射时的角度和速度，保证导弹命中目标
b. 发射向任意一个方向即可，飞上天就行，每隔一小段时间计算与目标的误差，根据误差调整飞行方向，不断循环这个过程，直至命中目标

显然，我在通往目标的道路上没有采用 a 方法的能力，只能采用 b 方法了。回顾就是就是计算我与目标的误差，那我下一次的计划就是在调整飞行方向了。

c. 不要再精疲力尽的时候回顾，你只会是想早点结束它而已，可以在每天一个固定的时间回顾，也不必等全部任务都完成再做
d. 如果有时间记录的话，做起回顾事半功倍

(3) 、聊聊自己为什么坚持做回顾

1.1重头再来

在2012年下半年在战拖学园和一群人天天刷番茄，那一段时间学到了很多东西。也从零散中看了不少东西，看到什么就跟着试试，包括那个四点起床，结果被骂了一顿后就放弃了。从记录看到了十一月我就开始放弃了。每次看到群里的热闹，感觉自己已经像逃兵一样放弃了他们，再加入也不好意思。中途也试着想再加入，没有一周的时候，因各种原因放弃（现在看来如果我多坚持一下，不太在意亲友的想法，也不至于这样被动。当然半途而废一直以来是我最大的问题）。作为逃兵我，都不敢面对之前的战友。三年间除了看看李公子的学习力外，基本都没有怎么过多关注个人管理。8月17日老琼在拖学园番茄土豆群中说要再次来战拖，看到之前失败战友都能勇敢去面对过去，再次站起来，为什么我不能呢?于是准备开始番茄工作法学习打卡,一切头重再来。
就在那几天，开始再次关注幸福进化群，记得之前还是零言向我推荐村长的博客和群。现在想想总会有点辜负了她的信任。在一天看在群里看到一个人问excel日期分组的问题，看着没有人回说，我就帮忙了，在不清楚那人具体想干什么时候的情况下，我帮了一下倒忙。加上当时网卡，基本不能远程，我就给那人录制了几个gif图片演示。达到那人效果时，他说怎么和剑飞的那个怎么不样…给文档…再折腾…折腾了几个来回才帮助搞好。如果是个会Excel的人肯定会说我浪费时间了,好在他不会excel…..机缘巧合，交流之后他告诉我他参加过3721，鼓励我参与，也许算是回报，他还热情说把3721的pass卡给我,让我直接去参加。我看看第7期还有一段时间，就说我先看看能不能报上。结果，在我以为报不上的时候，居然收到了短信。

1.2后起之秀

就像夏花问我知不知道吸引力法则时,我当时就明白她想说什么。也许是那个吸引力，也许是我运气好。在那几天宠辱不惊通过群问我一个问题。在交流，我发现她身上充满激情，虽然在个人成长方面知道的不多，但她无意中说到：她想看看她坚持学习五年后的自己是什么样子的。三年前我也像她一样，渴望学习，也希望看看几年后的自己，可惜半途而废了几年后的自己让自己很难堪。她无意间的一句话把我自己心里那一种沉睡中的渴望点起来。刚好通过这一次3721，我可以认识一些一起成长的人，可以相互学习一下也让自己不要放弃，学会每天计划与回顾。

2.初印象

2.1简单又有难度

在接到邮件时看要求时的第一映象是简单而又有压力：

简单在于就三件事，没有什么要求，要求主要是打卡时间。
压力方面：

  • 一方面是每天早上8：00前打卡。这个要是周一和周末有点麻烦，周一基本上5点半就起了，洗漱完就出门坐车去了。到公司都过了8点。周末不太不确定，平时都是自然醒的。有事就早点，没事就迟一些。
  • 二是动不动的就是红包。对于几乎不用微信的我生怕不小心点错了什么。
  • 三是组队后的同组的压力，同伴都很优秀，自己都不好意思放弃。

2.2 第一次畅聊

在第一次畅聊中，我发现伙伴们都很优秀，来迟的我差点以为没有人要了，感觉@阿甲还是挺淡定的。
通过第一次的了解，发现有不少人都是之前参加过的，大家来自不同的地方和专业方向，但都是一样的追求和渴望成长。
交流中印象最深的是马博士说的“一件事情，你对它有多大的期待，就要投入多少的时间和精力，只有高投入才会达成你的期待~”，
然后就是蒋金涛说的3个目标：
1.努力规范自己养成一个好的生活工作习惯 早睡早起 按时吃饭 身体健康；
2.每天努力完成计划 每天能感到充实 成长 快乐；
3.认识更多优秀的朋友影响自己变成更好的自己。

3.收获

3.1学到不少东西

看着别人打卡的信息中，无意中可以学到很多东西。

3.2关于回忆

口号:当时组队后说口号时，我无意中想到当时高考时班里每天去跑步时的口号“逆风飞扬,永不止步”，每次回想，会让自己有种兴奋的感觉。所幸大家也采纳了。To be no.1当时和队友说是刘德华唱的残奥会的歌，结果当晚去找了找没有找到，才发现名字记错了那歌是《Everyone is NO.1》，记得当时看这个MV时，感触很深，总之无论怎样都不要放弃。
关于扇贝：在采集村长博客时，看到村长也提到了Chain，想想估计当时我们是一个时期看到扇贝周刊采访Chain的文章。12年看到Chain介绍的我当时正在天天战拖，当时就各种跟。四点起床是第一次在他这里听到的，之后还看到还有一本《四点起床》的书，结果在我折腾一二周被骂了一顿就放弃。在看到文章后，我们都在学，可惜扇贝一两个月后，我随着停止了战拖，扇贝也跟着放弃了。如果三年我也像村长这样学，我现在会不会感谢自己，而不是自己自责。三年的教训后的总结是：半途而废是最浪费时间的！也许像笑来老师说的那个，有些事说给经历过的人，才更有感受（他是说书准备写给学生看的，结果是工作过的人看的更多）。
关于一万小时：在再次听到这本书时，第一反应就是想到@ Hamlet当时组织的一万小时的活动，可惜当时没有参加。天天看到他那样坚持，感觉很内疚的。我什么时候才能开始自己的一万小时的精深练习呢？

3.3从失败中看到自己的成长

“今天早6点半本来应该去跑步,忙完日志和计划后都40了.平时跑步洗澡洗衣就要近1小时,感觉今天很赶,想用这时间来看书做一下笔记走路听更好一点.中途强制休息时开始看手机资讯起来.然后就到了快上班时,结果没有跑步也没怎么学习,就感觉很失落.好在上班路上听书,感觉就好了,造成这种失落的关键问题在哪?”当时这个问题感谢朋友们给我了很多建议。其中一个朋友对我说，你的晨间日记还没有断，我突然发现还真是那么一回事。
虽然现在问题还有很多，但一些事还在坚持着好，如果没有时间做就少做一点。有些习惯一断就很难再起来。突然想到索隆回鹰眼“你为什么不躲？”，索隆说：“我不知道……我自己也不知道啊。我只是觉得如果在这儿退缩了，过去那些重要的誓言，约定全都会消失不见。而我再也回不到这里。”可惜我不做到。越写越像给未来放弃的自己看的，如果你看到，希望你不要想太多，重新站起来！现在的我给你一点积极的回忆。未来还是要靠你来走。

4.21天中的思考

4.1 三件事写什么?

到现在为止我也还没有想清楚这个问题？懵懵懂懂的感觉要看你自己的目的和场景。这个问题没有所谓的对与错，只有合不合适。
经验

  1. 青蛙模式：第一次畅聊时@雷慧洁(甜甜)说的：“三件事的选择是有一定依据地，就是一天最重要的事，是那种你不做有可能就会死的事情！“感觉对现在的我感觉很重要，我现在经常不能做到要事第一。
  2. 培养型：这是一次夏花对我说，三件事是两件最大青蛙，一件是与成长相关的。她的想法是怕自己忘记了成长。
  3. 轻松模式：三件小事开始，培养习惯开始做法。
  4. 乱七八槽：估计我算是这一种吧，感觉我很乱,按自己的想法瞎改。不过这样应该会更加有趣一点。

    战拖和3721的重叠，因为每天都要把计划和回顾发给别人，还要加入到doit.im。开始只是想加个标签（看到剑飞时间统计里面的提到），我的excel表是为了战拖番茄记录与回顾用的。为了让excel合成一个格式,给3721和doit用。开始想最重要三件事，像夏花那样。但每次排序感觉很纠结，为了简单，直接按我的时间来，前三件事就成三件事，一般都是早上做的三件。最近感觉自己的不能做到要事第一。需要每天先分配最重要的事，然后再按时间来排序。前天早上打坚持卡时，发现之前只做了21天的格式。就干脆直接在excel中合成一个3721的格式,之后早上就可以不用列格式让excel自动生成。今天早上无意看到excel的表格复制到微信中直接是图片。以后就是直接发图，格式问题就不再用考虑算了。回顾也一样可以生成一个,并列看结果和思考也不错,反正没有人在意我的格式。
